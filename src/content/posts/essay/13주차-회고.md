## 배운 것

1. **Flyway**: DB 스키마를 코드로 버전 관리. `V{번호}__{설명}.sql` 네이밍이 핵심이고, 한 번 적용된 마이그레이션은 수정 불가.
2. **ArchUnit**: 아키텍처 규칙을 테스트로 강제. controller → repository 직접 접근 같은 위반을 CI에서 잡을 수 있다.
3. **JPA Auditing 함정**: `@DataJpaTest`에서 `@Configuration`은 자동 스캔이 안 돼서 `@Import` 필수. `entityManager.clear()` 빠뜨리면 1차 캐시에서 반환되어 audit 검증이 의미 없어진다.
4. **Port/Adapter 패턴**: 외부 API 의존을 인터페이스 뒤에 숨기면 WireMock으로 깔끔하게 테스트할 수 있다.
